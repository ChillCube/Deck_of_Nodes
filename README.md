# Deck of Nodes

An addon for managing a list of nodes. Useful for card games

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/Deck_of_Nodes.git
```

## Documentation
https://github.com/ChillCube/Deck of Nodes/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon Deck of Nodes
```
